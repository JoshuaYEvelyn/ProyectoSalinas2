TIPOS DE FRAMEWORK

Frameworks WEB

Estos frameworks se utilizan para desarrollar aplicaciones web y ofrecen una estructura para la lógica de negocio, la presentación y la interacción con el usuario. Algunos de los más populares son:

  • Frameworks de front-end: Se encargan de la interfaz de usuario y la interacción con el navegador. Ejemplos: React, Angular, Vue.js.
  • Frameworks de back-end: Manejan la lógica del servidor y la base de datos. Ejemplos: Django (Python), Laravel (PHP), Express.js (Node.js).
  • Frameworks full-stack: Combinan características de ambos, ofreciendo una solución completa para el desarrollo web. Ejemplos: Ruby on Rails, MEAN Stack (MongoDB, Express.js, Angular, Node.js)

Frameworks Móviles

Diseñados para crear aplicaciones móviles nativas o híbridas. Algunos ejemplos son:

  • Nativos: iOS: Swift UI, UIKit, Android: Jetpack Compose, Android View System
  • Híbridos: React, Native, Flutter, ,Ionic

Frameworks de Testing

Facilitan la creación y ejecución de pruebas automatizadas para asegurar la calidad del software. Ejemplos:

  • Unit testing: JUnit, NUnit
  • Integración: Selenium, Jest
  • Performance: JMeter, Gatling

Frameworks de Microservicios

Facilitan el desarrollo de aplicaciones divididas en pequeños servicios independientes. Ejemplos:

  • Spring Boot
  • Node.js con Express.js
  • Go Kit
